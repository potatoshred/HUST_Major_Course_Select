## 使用方法

1. 配置python3环境，并 `pip install requests`
2. 下载[本脚本](./selectMajor.py)，手动配置**Cookie**和**待选课程**，User-agent一般**不需要**修改.
3. 待选课系统打开后，运行 `python selectMajor.py`
